Projects

A selection of things I've built — from backend pipelines to full-stack apps.

Rental Car Analytics Pipeline · Priceline

During my co-op at Priceline, I built a real-time data pipeline that consumed rental car booking events from Apache Kafka and loaded them into Google BigQuery. The pipeline processed over 1.2 million events per day and served dashboards used by 90+ team members. I also built validation layers to handle schema inconsistencies across suppliers, which significantly improved data quality downstream.

Tech stack:

  • Apache Kafka · Google BigQuery · Java / Spring Boot · SQL

Social Equity Action Lab Website · SEAL @ Northeastern

I'm currently building and maintaining the website for SEAL, a research lab at Northeastern focused on technology and social justice. The site uses Next.js 14, TypeScript, Tailwind CSS, Zustand for state management, and Framer Motion for animations. It's been a great way to deepen my frontend skills while contributing to work that actually matters.

Tech stack:

  • Next.js 14 · TypeScript · Tailwind CSS · Zustand · Framer Motion
